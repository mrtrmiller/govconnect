# How to upload GovConnect v2 to GitHub

1. Download and unzip `govconnect-v2.zip`.
2. Open your GitHub repo: `https://github.com/mrtrmiller/govconnect`.
3. Use **Add file → Upload files**.
4. Drag the contents of the unzipped `govconnect-v2` folder into GitHub.
5. Commit the changes.
6. Wait about a minute for GitHub Pages to refresh.
7. Open: `https://mrtrmiller.github.io/govconnect/`.

## Important

This version reorganizes the repo. Your old AP Gov app moves into the `ap-gov/` folder, and the main `index.html` becomes the landing page.

If GitHub asks whether to replace existing files, that is expected for `index.html` and `README.md`.
