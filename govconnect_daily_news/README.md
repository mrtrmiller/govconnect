# GovConnect Daily News

A static daily news classroom site for AP Government, Government/Civics, Economics, and U.S. History.

## Sunday update workflow

Edit these files only:

- `data/apgov.json`
- `data/gov.json`
- `data/econ.json`
- `data/ushistory.json`

For each weekday, replace:

- `title`
- `summary`
- `tags`
- `vocabulary`
- `questions`
- `written.prompt`
- `exitTicket`

Do not change the brackets, commas, or quotation marks.

## Deployment

Upload the whole folder to GitHub and deploy with GitHub Pages, Netlify, or Vercel.
