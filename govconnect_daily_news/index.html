Sunday Update Workflow

Each course has one editable JSON file in the /data folder. Update the five daily stories before the week begins, commit to GitHub, and the site refreshes automatically after deployment.

  1. Open /data/apgov.json, gov.json, econ.json, or ushistory.json.
  2. Replace the headline, summary, vocabulary, questions, and written response prompt.
  3. Keep the same quotation marks, commas, and brackets.
  4. Commit changes to GitHub. Your published site updates through GitHub Pages/Netlify/Vercel.

Teacher tip: Keep the summary factual. Let students make the course connection through the questions.
